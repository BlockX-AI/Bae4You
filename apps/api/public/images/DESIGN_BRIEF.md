# Bae4U NFT Image Assets - Design Brief

**Project:** Bae4U SocialFi Dating App - NFT Image System
**Deliverable:** 10 static image assets for NFT metadata
**Format:** PNG with transparency
**Timeline:** ASAP

## Overview

Bae4U is a blockchain-based dating app where users can mint NFTs representing their profiles, hero cards, achievement badges, and couple cards. We need 10 static image assets that will be used as frames and badges for dynamically generated NFT images.

## Brand Identity

- **Primary Color:** Indigo/Purple (#6366f1)
- **Secondary Colors:** Pink, Red for romantic elements
- **Style:** Modern, clean, premium, slightly playful
- **Target Audience:** Gen Z/Millennial dating app users
- **Vibe:** Fun, social, premium, collectible (like trading cards)

## Technical Specifications

| Asset Type | Dimensions | Format | Notes |
|------------|-----------|---------|-------|
| Card Frames | 400x400 px | PNG | Transparent center (circle or square) |
| Badge Images | 200x200 px | PNG | Transparent background |
| Resolution | 72 DPI minimum | - | Higher is better for quality |

**Important:** All card frames must have a **transparent center area** where user photos will be dynamically inserted.

## Design Guidelines

### Consistency
- Maintain consistent design language across all assets
- Use similar line weights and proportions
- Keep the same overall aesthetic (modern, clean, premium)
- Ensure all frames have the same transparent center size

### Color Palette
- **Common:** Grey (#808080, #A0A0A0)
- **Rare:** Blue (#00BFFF, #1E90FF, #4169E1)
- **Epic:** Purple (#9400D3, #8A2BE2, #9932CC)
- **Legend:** Gold (#FFD700, #FFA500, #DAA520)
- **Couple:** Pink/Red (#FF69B4, #FF1493, #DC143C)
- **Bronze:** (#CD7F32, #B87333)
- **Silver:** (#C0C0C0, #E8E8E8)
- **Gold:** (#FFD700, #FFC125)
- **Diamond:** Crystal with rainbow reflections
- **Master:** Rainbow gradient (#FF0000, #FF7F00, #FFFF00, #00FF00, #0000FF, #8B00FF)

### Typography (if text is needed)
- Font: Modern sans-serif (Inter, Roboto, or similar)
- Weight: Bold or Extra Bold
- Style: Clean, legible

### Effects
- **Glow:** Subtle, not overwhelming
- **Shine:** Realistic metallic reflections
- **Sparkles:** Small, scattered, not distracting
- **Gradients:** Smooth, professional color transitions
